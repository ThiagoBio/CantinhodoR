####Cores no R: uma lista imensa.

[A lista está aqui, é só clicar e ser feliz.](http://www.ufpa.br/dicas/htm/htm-cor2.htm)

No R, você pode definir qualquer cor usando o nome entre aspas, é bem fácil. Só para dar um exemplo, veja o gráfico abaixo, no qual usei alguams cores escolhidas arbitrariamente pela lista:

```{r}
cores<-c("Gainsboro", "Lavender", "DeepSkyBlue", "LawnGreen", "Gold",
         "MediumOrchid","LightSalmon", "Sienna", "Tomato", "DeepPink1" )

barplot(1:10, col=cores)
```

É isso aí. Agora você pode usar aquele tom super específico de violeta claro que sempre teve vontade de colocar no gráfico mas não sabia como. :)
